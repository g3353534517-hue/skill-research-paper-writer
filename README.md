# Research Paper Writer / 学术论文写作助手

![分类](https://img.shields.io/badge/分类-写作-green)

## 项目简介

学术论文全流程写作技能。从选题到终稿，覆盖论文写作的每个阶段。

## 支持阶段

1. 选题与文献综述
2. 研究方法设计
3. 数据分析与结果呈现
4. 讨论与结论撰写
5. 格式规范与引用检查

## 写作原则

- 学术严谨：观点有据，引用规范
- 逻辑清晰：论点-论据-论证链条完整
- 语言规范：学术文体，避免口语化
- 原创性：强调独立思考，标注所有引用来源
